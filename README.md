# Voxel Cone Tracing

![alt text](https://raw.githubusercontent.com/cheapbrain/gidemo/master/screenshots/1.jpg)

**Features:**
- Diffuse Cones
- Specular Cones
- Shadow Cones
- Emissive materials
- HDR RGBA16f storage
- Anisotropic voxels
- Temporal light multi-bounce
- Particle systems

OpenGL 4.6 required
